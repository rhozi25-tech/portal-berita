== Description ==

The *ThemeZee Toolkit* is a collection of useful small modules and features, neatly bundled into a single plugin. This plugin has been designed specifically for ThemeZee WordPress themes but will also work with any theme.

= Included Features =

* Widget Visibility - Hide or show widgets conditionally
* Infinite Scroll - Add support for infinite scrolling on the blog homepage

= Documentation =

For detailed setup instructions, visit the official [Documentation](https://themezee.com/docs/toolkit-documentation/) page

== Installation ==

1. Activate the plugin
2. Go to Settings > ThemeZee Plugins > Toolkit and enable all the modules you need
3. For detailed setup instructions, visit the official [Documentation](https://themezee.com/docs/toolkit-documentation/) page
